# ssh-bruteforcer
A simple ruby tool for brute forcing ssh port on a website

# Steps To Run This Tool:
- git clone https://github.com/krishpranav/ssh-bruteforcer
- cd ssh-bruteforcer
- sudo chmod +x *
- sudo bash install.sh
- ruby ssh_bruteforcer.rb
